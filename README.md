# Password Generator

A Python program that generates secure random passwords.
It allows users to specify the number of passwords and their length,
validates input to prevent errors, and copies generated passwords to the clipboard.

## Features

- Generate multiple passwords at once
- Set the length of each password
- Input validation to prevent invalid entries
- Automatically copies each password to the clipboard

## Requirements

- Python 3.14
- pyperclip library ('pip install pyperclip')

## How to run

1. Clone the repository
2. Install required packages: 'pip install pyperclip'
3. Run the program 'python password_generator.py'
4. Follow the prompts to generate passwords
